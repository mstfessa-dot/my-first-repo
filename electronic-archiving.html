Electronic Archiving System

Client: King Abdulaziz University

Role: Project Coordinator / Systems Analyst

Type: Digital Transformation Project

Project Background

The university was operating multiple legacy systems across different departments. Data and documents were fragmented, making integration and digital transformation extremely challenging.

The Challenge

  • Multiple outdated and incompatible systems
  • Difficulty accessing key decision makers
  • Strong competition from well-established global vendors
  • High resistance to introducing a new system

My Role

  • Analyzed the university’s existing systems landscape
  • Reached and engaged decision makers through multiple meetings
  • Secured approval to execute a pilot project
  • Coordinated with outsourced engineers from the vendor’s parent company
  • Supervised pilot implementation and system integration
  • Supported the project during the financial tender phase

Outcome

  • Successful pilot execution
  • Proof of system compatibility and integration capability
  • Project qualified for the official competitive tender stage
